# Eternal Infinite Void

Personal portfolio website built as a JavaScript and React app. The site combines an interactive black hole intro, responsive portfolio sections, animated canvas particles, and expandable project cards for software, AI, embedded systems, PCB design, and creative technical work.

The React application lives in [`my-app`](./my-app).

## Live Site

https://isaacmatt.github.io/eternal-infinite-void

## Highlights

- Interactive black hole animation with hover, click, keyboard, and video playback effects
- Canvas particle background that responds to pointer movement
- Expandable work cards with categories, tags, descriptions, and repository links
- Project coverage across React, AI, machine learning, embedded systems, Raspberry Pi Pico W WiFi work, and SD card PCB design
- GitHub Pages deployment workflow

## Tech Stack

- JavaScript
- React
- CSS
- Create React App
- HTML canvas
- GitHub Pages

## Running Locally

```sh
cd my-app
npm install
npm start
```

The development server opens at:

```text
http://localhost:3000
```

## Building

```sh
cd my-app
npm run build
```

This creates the production build in `my-app/build`.

## Deployment

The app is configured for GitHub Pages through the `homepage` field in `my-app/package.json`.

```sh
cd my-app
npm run deploy
```

This project was developed through an iterative frontend workflow using JavaScript and React, with AI-assisted collaboration. Claude was part of the creative and planning process for interface ideas, copy refinement, and interaction design decisions.
