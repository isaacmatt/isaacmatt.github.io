## About This Repository

This repository is my personal portfolio website — a JavaScript and React app. The site combines an interactive black hole intro, responsive portfolio sections, animated canvas particles, and expandable project cards for software, AI, embedded systems, PCB design, and creative technical work.

The React application lives in [`my-app`](./my-app).

## Highlights

- Interactive black hole animation with hover, click, keyboard, and video playback effects
- Canvas particle background that responds to pointer movement
- Expandable work cards with categories, tags, descriptions, and repository links
- Project coverage across React, AI, machine learning, embedded systems, Raspberry Pi Pico W WiFi work, and SD card PCB design
- GitHub Pages deployment workflow

## Running Locally

```sh
cd my-app
npm install
npm start
```

The development server opens at:

```text
http://localhost:3000
```

## Building

```sh
cd my-app
npm run build
```

This creates the production build in `my-app/build`.

## Deployment

The app is configured for GitHub Pages through the `homepage` field in `my-app/package.json`.

```sh
cd my-app
npm run deploy
```
